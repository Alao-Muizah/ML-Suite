# ML-Suite Dashboard 

ML-Suite is a curated collection of machine learning projects built to solve real-world prediction, classification, and analytics problems. Each project is deployed as an interactive application and accessible through a unified web dashboard.

The suite focuses on practical applications of machine learning, including feature engineering, predictive modeling, and data-driven decision systems.

---

Access all deployed projects from a single interface:

## Projects Included

- **CO₂ Emission Predictor**  
Estimates vehicle emissions using fuel consumption and vehicle features.  
→ [*View project repo*](co2-emission-model)


- **Electricity Forecasting**  
Forecasts energy usage using lag features, rolling statistics, and cyclical encoding.  
→ [*View project repo*](electricity-prediction-model)

- **Customer Churn Prediction**  
Predicts customer retention based on behavioral and transactional patterns.  
→ [*View project repo*](churn-prediction-model)

- **Tutor Recommender System**  
Recommends tutors based on learning goals and user preferences.  
→ [*View project repo*](tutor-recommender)

## Architecture

Users interact with a central dashboard that links to individual deployed Streamlit applications for each machine learning model.

Dashboard → Streamlit Apps → ML Models
